# Troubleshooting the Blog

I’m really enjoying this workflow I built to blog more. What I do is go to my site [https://prabhchintan.com/post](https://prabhchintan.com/post) which is where I am posting this from, write a few lines and pick a slug, then input an authentication token to post it.

I am still troubleshooting it and this is a test post to see if a change I just made worked. It’s to better update the site automatically as soon as I post, whereas I had to manually update it before.

Future ideas I want to implement are posting media (pictures, videos, music, etc.), maybe concurrently post it on social media (for now I only use X), and add a comment section.
